Zomato Data Analysis Project
📌 Project Overview

This project focuses on analyzing Zomato restaurant data to discover meaningful insights about restaurant ratings, cuisines, pricing, customer preferences, and location trends using Python.

The project was completed using Google Colab and includes:

Data Cleaning
Exploratory Data Analysis (EDA)
Data Visualization
Business Insights & Recommendations

🎯 Objectives
Analyze restaurant ratings and customer preferences
Identify popular cuisines and restaurant locations
Study the relationship between pricing and ratings
Explore online ordering and table booking trends
Generate insights useful for food delivery platforms


📂 Dataset Information

The dataset contains restaurant-related information such as:

Restaurant Name
Location
Cuisine Types
Ratings
Votes
Approximate Cost
Online Order Availability
Table Booking Availability

Dataset Format:

CSV / Excel File


🛠️ Technologies Used
Python
Google Colab
Pandas
NumPy
Matplotlib
Seaborn

1. Data Collection

2. Data Cleaning
